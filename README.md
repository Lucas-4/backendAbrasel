# Backend usado para o teste #3 da Abrasel

## Como usar 

* Clone o repositório

```
git clone https://github.com/Lucas-4/backendAbrasel.git
```

* Vá para o diretório do repositório
```
cd backendAbrasel
```

* Instale as dependências

```
npm install
```

* Execute a aplicação

```
npm start
```
